# 🎲 TableGamesBot

> Чтобы каждая партия состоялась.

## 🌟 О чём этот проект

Мы любим настольные игры. Но не любим тот момент, когда договориться собраться сложнее, чем выиграть в «Цивилизацию». TableGamesBot создан, чтобы убрать барьеры между желанием поиграть и самой игрой.

Бот автоматизирует всё, что обычно отнимает силы и время:
- сбор игроков
- выбор игры, даты и места
- напоминания и подтверждения
- управление листом ожидания

Проект родился в колледже, делается с душой и остаётся полностью бесплатным.

### 📦 Быстрый старт

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/NikkiShuRA/TableGamesBot.git
   cd TableGamesBot
   ```
2. **Установите зависимости:**
    ```bash
    npm install
    ```
3. **Настройка окружения:**

   Создайте файл .env в корне проекта:
   ```env
   # --- Telegram Bot Настройки ---
   # Получить у @BotFather
   BOT_TOKEN=123456789:ABCDefGhIJKlmNoPQRstUVwxYZ
   
   # --- Supabase Настройки ---
   # Найти в Project Settings -> API
   SUPABASE_URL=https://your-project-id.supabase.co
   SUPABASE_KEY=your-supabase-anon-key
   
   # --- Настройки логирования (Pino) ---
   # Уровень логирования: info, debug, production
   LOG_LEVEL=info
   ```
4. **Запуск:**

    Бот:
    ```bash
    npm run dev
    ```
    Тест:
    ```bash
    npm run test
    ```
